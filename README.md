# E-Commerce-analysis
# E-Commerce Customer Behavior and Sales Analytics

## Project Overview
This project focuses on analyzing e-commerce customer behavior, product sales, and purchasing trends using Python, Power BI, and data analytics techniques. The objective is to generate meaningful business insights from retail transaction data and visualize them through interactive dashboards.

---

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- Google Colab
- GitHub

---

## Dataset
The dataset contains:
- Customer information
- Product details
- Order history
- Payment methods
- Review scores
- City-wise sales data
- Purchase dates and times

Dataset Source:
https://www.kaggle.com/datasets/ertugrulesol/online-retail-data

---

## Data Cleaning Process
Performed data preprocessing and cleaning using Python:
- Removed missing values
- Removed duplicate records
- Converted data types
- Created TotalPrice column
- Extracted year, month, day, and order hour
- Prepared dataset for visualization and analysis

---

## Exploratory Data Analysis (EDA)
Analyzed:
- Monthly sales trends
- Top-selling products
- Revenue by category
- Customer purchasing behavior
- Payment method usage
- City-wise sales distribution
- Review score analysis

---

## Power BI Dashboard Features
Created an interactive dashboard containing:
- Total Revenue KPI
- Total Orders KPI
- Total Customers KPI
- Monthly Sales Trend
- Category-wise Revenue Analysis
- Customer Demographics
- Payment Method Analysis
- Top Products Visualization
- Interactive Filters and Slicers

## Business Insights
- Identified high-revenue product categories
- Analyzed customer purchasing patterns

- Compared sales performance across cities
